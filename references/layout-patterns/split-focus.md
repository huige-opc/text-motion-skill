# split-focus 左右焦点布局

用于解决 16:9 画面左侧堆内容、右侧空置的问题。适合 dense/balanced 场景、观点解释、流程拆解、代码/工具说明。

## 结构

- 左侧 42%-48%：主标题、短说明、关键词标签。
- 右侧 34%-42%：辅助面板、步骤卡、代码/流程/图形结构。
- 中间可用细线、导轨、箭头或编号连接左右信息。

## 规则

- 左右两侧都必须有信息或明确视觉功能；右侧不能只是装饰空白。
- 主标题最多 2-3 行，右侧面板最多 3 个信息点。
- dense 场景减少背景装饰，优先强化左右关系。
- tech 主题右侧面板用低透明边框和细线，不用大面积霓虹 glow。

## 常见用法

- 左：一句结论；右：3 个原因。
- 左：问题；右：解决流程。
- 左：核心概念；右：代码/工具/组件结构。

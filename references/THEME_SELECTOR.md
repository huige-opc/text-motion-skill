# 主题自动选择逻辑

## 概述

当用户未明确指定主题时，根据 SRT 内容自动匹配最佳主题。

所有 31 个主题（9 个基础 + 22 个扩展）统一存放在 `themes/` 目录下，参与平等的关键词评分竞争。

## 选择流程

```
1. 读取 SRT 内容
2. 对每个主题计算关键词匹配得分
3. 按得分排序，最高分 = 主主题
4. 返回主主题 + 2 个备选
```

## 返回结构

```json
{
  "primaryTheme": { "id": "tech", "confidence": 0.72 },
  "alternatives": [
    { "id": "data-grid", "confidence": 0.18 },
    { "id": "ai-console", "confidence": 0.10 }
  ]
}
```

## 命令行

```bash
# 自动选择
node select-theme.mjs audio.srt

# 显式指定主题
node select-theme.mjs audio.srt --theme cyber-neon
```

# 场景骨架：两栏通用（Scene Two Column）

左右两栏并列内容的通用布局。

## 适用场景

内容并列展示、分类列举、特征对比。

## 布局示意

### Pure-Graphic（全屏版）

```
┌──────────────────────────────────┐
│  grid-bg + floating-dots         │
│                                  │
│        [●] 眉标文字                │
│                                  │
│  ┌──────────┐  ┌──────────┐    │
│  │  左栏标题  │  │  右栏标题  │    │
│  │          │  │          │    │
│  │  左栏内容  │  │  右栏内容  │    │
│  │          │  │          │    │
│  └──────────┘  └──────────┘    │
│                                  │
│          底部总结                 │
└──────────────────────────────────┘
```

### Talking-Head（真人版）

```
┌──────────────────────────────────┐
│ [●] 眉标文字                       │
│                                  │
│  ┌──────┐  ┌────────────────────┐│
│  │左栏   │  │                    ││
│  │内容   │  │    [真人]           ││
│  │       │  │                    ││
│  │       │  │                    ││
│  └──────┘  └────────────────────┘│
│          底部总结条                │
└──────────────────────────────────┘
```

## 组件清单

| 类别 | 组件 | 说明 |

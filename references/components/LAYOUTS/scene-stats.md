# 场景骨架：数据证明（Scene Stats）

展示数字指标和统计数据的数据驱动场景。

## 适用场景

视频中段，用数据佐证观点。

## 布局示意

### Pure-Graphic（全屏版）

```

│  grid-bg + floating-dots         │
│                                  │
│        [●] 先给你几个数字            │
│       这个 skill 叫 text-motion    │
│                                  │
│  ┌──────┐  ┌──────┐  ┌──────┐  │
│  │  9   │  │  5   │  │  6   │  │
│  │套视觉  │  │类场景  │  │道步骤  │  │
│  │风格   │  │类型   │  │自动   │  │
│  └──────┘  └──────┘  └──────┘  │
│                                  │
│  ▓▓▓▓▓▓▓  ▓▓▓▓▓   ▓▓▓▓          │

```

## 典型时间编排

| SRT beat | 揭示内容 | 时长 |
|----------|---------|------|
| 1 | 背景网格 + 装饰点 + 眉标 + 标题 | 0–2.9s |
| 2 | 卡片1：弹性弹入 + 条 + 环 + 进度 + 星 | 2.9–5.8s |
| 3 | 卡片2：同上 | 5.8–8.6s |
| 4 | 卡片3：同上 + 连接线 | 8.6–11.5s |
| 5 | 底部说明 + 柱状图 + 强调条 | 11.5–14.4s |

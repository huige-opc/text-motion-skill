# 场景骨架：痛点开场（Scene Hook）

视频开头的痛点/问题引入场景。

## 适用场景

视频第 1-2 帧，列举用户痛点，引发共鸣。

## 布局示意

### Pure-Graphic（全屏版）

```
┌──────────────────────────────────┐
│  grid-bg + floating-dots         │
│                                  │
│        [●] 做视频的你 · 是不是也这样    │
│                                  │
│  01 口播录好了 配画面怎么办 ?  [标签]  │
│  02 打开剪辑软件 一堆轨道   ?  [标签]  │
│  03 从哪下手     都不知道 ?  [标签]  │
│                                  │
│  ◄ 3 个痛点 ─────────────── 100% ► │
└──────────────────────────────────┘
```

### Talking-Head（真人版）

```
┌──────────────────────────────────┐
│ [●] 做视频的你 · 是不是也这样        │
│                                  │
│  ┌─────┐     ┌────────────────┐ │
│  │01 口播│     │                │ │
│  │02 剪辑│ [真人]  画面区域       │ │
│  │03 下手│     │                │ │
│  └─────┘     └────────────────┘ │
└──────────────────────────────────┘
```

## 典型时间编排

| SRT beat | 揭示内容 | 时长 |
|----------|---------|------|
| 1 | 背景装饰 + 眉标 | 0–1.6s |
| 2 | 行1：角标+文本+问号+标签+下划线 | 1.6–3.3s |
| 3 | 行2：同上 + 背景形状 | 3.3–4.9s |
| 4 | 行3：同上 | 4.9–6.6s |
| 5 | 底部进度条 + 装饰环 | 6.6–8.2s |

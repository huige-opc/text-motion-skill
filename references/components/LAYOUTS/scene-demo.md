# 场景骨架：流程演示（Scene Demo）

展示多步骤工作流的演示场景。

## 适用场景

功能介绍、流程演示、步骤拆解。

## 布局示意

```
┌──────────────────────────────────┐
│  grid-bg + floating-dots         │
│                                  │
│        [●] 三步搞定                │
│                                  │
│    [01 输入]  [02 处理]  [03 输出]  │
│    ─────────────────────────        │
│                                  │
│          ┌──────────────┐        │
│          │  当前步骤内容  │        │
│          │  ┌────────┐  │        │
│          │  │ mock UI │  │        │
│          │  │ 演示区域  │  │        │
│          │  └────────┘  │        │
│          └──────────────┘        │
│                                  │
│      ▓▓▓▓▓▓▓▓▓▓░░ 已用时 3s       │
└──────────────────────────────────┘
```

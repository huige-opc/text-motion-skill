# 场景骨架：方案对比（Scene Solution）

Before vs After 左右对比场景。

## 适用场景

展示传统方式 vs AI/新方式的改进效果。

## 布局示意

### Pure-Graphic（全屏版）

```
┌──────────────────────────────────┐
│  grid-bg + floating-dots         │
│                                  │
│        [●] 以前 vs 现在            │
│                                  │
│  ┌──── 旧方式 ────┐  VS  ┌── 新方式 ─┐│
│  │ [标签] 传统流程  │      │ [标签] AI  ││
│  │ ┌────────────┐ │ ○    │ ┌────────┐││
│  │ │ 步骤1  ▓▓   ││      │ │ 步骤1 ✓ │││
│  │ │ 步骤2  ▓▓   ││      │ │ 步骤2 ✓ │││
│  │ │ 步骤3  ▓▓   ││      │ │ 步骤3 ✓ │││
│  │ └────────────┘ │      │ └────────┘││
│  │ ┌── 时间轴 ──┐ │      │ 自动完成    ││
│  │ │ V1 ▓▓     ││      │           ││
│  │ │ A1 ▓▓▓    ││      │           ││
│  │ └───────────┘│      │           ││
│  └───────────────┘      └───────────┘│
└──────────────────────────────────┘
```

### Talking-Head（真人版）

## 典型时间编排

| SRT beat | 揭示内容 | 时长 |
|----------|---------|------|
| 1 | 背景 + 眉标 | 0–2s |
| 2 | 左面板（旧方式）+ 标签 | 2–4s |
| 3 | VS 徽标 + 轨道动画 | 4–6s |
| 4 | 右面板（新方式）+ 标签 | 6–8s |
| 5 | 底部总结 + 强调条 | 8–10s |

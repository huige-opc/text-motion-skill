## 1. 审美分析维度

分析参考时，从以下 8 个维度提取特征：

### 1.1 色彩系统

```
主色调：
  - 最突出的 1-2 个颜色
  - 是冷色还是暖色
  - 是高饱和还是低饱和

辅助色：
  - 搭配主色的 2-3 个颜色
  - 渐变过渡方式

强调色/高光色：
  - 用于文字、装饰、关键元素的颜色

整体明度：
  □ 高调（明亮为主）
  □ 暗调（深色为主）
  □ 对比强烈（明暗分明）
  □ 柔和（中间调为主）

饱和度倾向：
  □ 鲜艳浓郁
  □ 柔和淡雅
  □ 性冷淡（低饱和）
```

### 1.2 光影质感

```
光照方向：
  □ 顶光 / 侧光 / 逆光 / 散射光

阴影软硬：
  □ 硬边阴影（直射光）
  □ 软边阴影（散射光）
  □ 无阴影（高调）

发光效果：
  □ 霓虹发光（Neon glow）
  □ 光晕效果（Bokeh/散景）
  □ 外发光（Outer glow）
  □ 无明显发光

渐变类型：
  □ 线性渐变（Linear gradient）
  □ 径向渐变（Radial gradient）
  □ 平滑渐变（Smooth transition）
```

### 1.3 构图布局

```
主体位置：
  □ 居中（Center）
  □ 三分法（Rule of thirds）
  □ 对角线（Diagonal）
  □ 黄金比例（Golden ratio）

留白程度：
  □ 紧凑（信息密集）
  □ 适度（呼吸感）
  □ 大量留白（极简）

层次感：
  □ 前中后景分明
  □ 景深虚化（浅景深）
  □ 平面化（无纵深）
```

### 1.4 字体风格

```
字体类型：
  □ 衬线体（Serif）— 传统、权威
  □ 无衬线体（Sans-serif）— 现代、简洁
  □ 等宽字体（Monospace）— 科技、代码
  □ 手写体（Script）— 亲和、手工

字重：
  □ 极细（100-300）
  □ 常规（400-500）
  □ 中粗（600-700）
  □ 极粗（800-900）

特殊效果：
  □ 倾斜（Italic/Skew）
  □ 描边（Stroke）
  □ 填充渐变（Gradient fill）
  □ 立体感（3D/Extrusion）
```

### 1.5 动效特征

```
转场速度：
  □ 快速（20-35帧）— 科技、时尚、活力
  □ 中等（35-50帧）— 商业、专业
  □ 慢速（50-70帧）— 自然、温暖、文艺

缓动曲线：
  □ linear（匀速）
  □ ease-out（减速收尾）— 最常用
  □ ease-in-out（头尾减速）
  □ backOut（弹性回弹）— 时尚、活力
  □ bounce（弹跳）

动效复杂度：
  □ 简单（单层动画）
  □ 中等（2-3层叠加）
  □ 复杂（多层复合）
```

### 1.6 质感纹理

```
背景质感：
  □ 纯色平滑
  □ 细微纹理（布纹/纸纹）
  □ 颗粒噪点（胶片感）
  □ 网格线（科技感）

文字质感：
  □ 平面无质感
  □ 轻微阴影
  □ 多层阴影（立体感）
  □ 发光质感

装饰元素：
  □ 几何图形（圆/方/三角）
  □ 线条/边框
  □ 粒子效果
  □ 光斑/散景
  □ 渐变遮罩
```

### 1.7 情绪氛围

```
整体情绪：
  □ 积极/活力
  □ 平静/专业
  □ 温暖/亲切
  □ 冷酷/科技
  □ 神秘/梦幻
  □ 怀旧/复古

风格关键词（3-5个）：
  1.
  2.
  3.
```

### 1.8 可复现特征总结

```
核心美学特征（3-5个）：
  1. [最突出的特征]
  2. [第二特征]
  3. [第三特征]

匹配主题：
  - 最适合的主题：tech / business / nature / fashion / warmth / professional / energy / artistic
  - 融合方案（如果需要）
```

## 2. 主题匹配关键词

### 8 主题核心关键词

| 主题 | 核心关键词 | 情感倾向 |
|-----|-----------|---------|
| **tech** | AI、数字、未来、智能、数据、代码、科技 | 科技感、未来感 |
| **business** | 增长、业绩、财富、品牌、发展、商业 | 专业、高端 |
| **nature** | 环保、绿色、生态、户外、阳光、生命 | 清新、自然 |
| **fashion** | 潮流、酷炫、街头、个性、态度、年轻 | 潮流、酷炫 |
| **warmth** | 爱、关怀、温馨、情感、回忆、家人 | 温暖、亲切 |
| **professional** | 咨询、教育、培训、知识、方案、方法 | 专业、清晰 |
| **energy** | 运动、突破、激情、挑战、冲刺、力量 | 活力、爆发 |
| **artistic** | 故事、情感、手工、文创、手绘、生活 | 文艺、复古 |

**注意**：即便选了 artistic / fashion / warmth 这些"复古 / 创意 / 温暖"的主题，**生成时仍要遵守 `AESTHETIC_GUARDRAILS.md` §5**。本节"核心关键词 + 情感倾向"只用于主题色 / 动效节奏选择，不代表要启用 deprecated 的纹理 / 转场 / 装饰伪标签。

### 关键词 → 主题 映射表

| 关键词 | 推荐主题 | 备选主题 |
|-------|---------|---------|
| AI/人工智能/数字化/代码 | tech | business |
| 增长/业绩/财富/品牌 | business | tech |
| 环保/绿色/生态/自然 | nature | warmth |
| 街头/潮流/个性/酷 | fashion | energy |
| 爱/关怀/温馨/家庭 | warmth | nature |
| 教育/培训/咨询/专业 | professional | business |
| 运动/挑战/激情/突破 | energy | fashion |
| 手工/文创/故事/文艺 | artistic | warmth |

### 情感倾向判断

```
正面/积极 → business, energy, warmth, nature
中性/专业 → professional, tech
深沉/情感 → artistic, warmth
酷炫/个性 → fashion, tech
```

| business | 深蓝+金色 | 中等、cubic、~~光泽~~ | 卡片级 backdrop-blur（局部）；**整屏 glass 禁用** |

| fashion | 黑+荧光粉+电光蓝 | 快、back、弹性 | 速度线（纯 SVG），~~动态模糊~~ |

实现标准（与 Remotion 教学视频基准对齐）：

- **每帧 ≤2 个信息组件**（不是 3）
- **文字层级 ≤3 层**（小标签 / 大标题 / 中文解读）
- **禁止 meta-pill / status-pill / tag / eyebrow-text 这类装饰文字**
- **禁止 glow / blur / glitch 堆叠**
- **场景转场默认 hard-cut，duration=0**
